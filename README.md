#### ``H i k s B O T``
---------
##### Termux Only
```bash
> pkg update && pkg upgrade
> pkg install git -y
> pkg install nodejs -y
> pkg install ffmpeg -y
> pkg install imagemagick -y
> git clone https://github.com/yahtoiba/hiks
> cd hiks
> npm install
```
--------

###### Run
```bash
> node . [<session name>] (session name is optional)
```

---------

###### End
```bash
> node index.js
```
